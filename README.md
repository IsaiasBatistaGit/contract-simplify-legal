# Contract Simplify - Legal Documents

This repository contains the legal documents for Contract Simplify, including the Privacy Policy and Terms of Service.

## Files

- `privacy.html` - Privacy Policy
- `terms.html` - Terms of Service

## GitHub Pages Setup

To enable GitHub Pages for this repository:

1. Go to the repository settings on GitHub
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select the branch (usually `main` or `master`)
4. Select the folder (usually `/ (root)`)
5. Click "Save"

Your pages will be available at:
- Privacy Policy: `https://isaiasbatistagit.github.io/contract-simplify-legal/privacy.html`
- Terms of Service: `https://isaiasbatistagit.github.io/contract-simplify-legal/terms.html`

## Updating Documents

To update these documents:

1. Edit the HTML files
2. Update the "Last Updated" date in each file
3. Commit and push the changes
4. GitHub Pages will automatically update (may take a few minutes)

## Notes

- These documents are designed to be mobile-friendly and accessible
- They use system fonts for optimal readability across devices
- The styling is minimal and professional
